# 界面美化更新说明

## 更新内容

### 1. 可配置的标题和介绍

在 `config.json` 中新增两个可选字段：

```json
{
  "title": "Scallop - 网络延迟监控",
  "description": "实时监控网络延迟，支持多目标对比分析",
  ...
}
```

- `title`: 页面标题，显示在浏览器标签和页面顶部（可选，默认为 "Scallop - 网络延迟监控"）
- `description`: 页面介绍，显示在页面顶部标题下方（可选，默认为空不显示）

### 2. 黑暗模式支持

新增主题切换功能，支持三种模式：

- **浅色模式**: 传统的白色背景界面
- **深色模式**: 护眼的深色背景界面
- **跟随系统**: 自动跟随操作系统的主题设置（默认）

主题选择会自动保存到浏览器本地存储，下次访问时自动应用。

### 3. 优化的图表显示

#### 标签式目标选择
- 所有监控目标以彩色标签形式展示
- 点击标签即可选中/取消选中该目标
- 选中的目标会在图表中实时显示
- 每个标签都有独特的颜色标识，与图表线条颜色对应
- 默认自动选中前3个目标

#### 图表特性
- 支持多目标同时对比显示
- 平滑的曲线动画效果
- 响应式设计，自适应不同屏幕尺寸
- 悬停显示详细数据点信息
- 支持 1小时/6小时/24小时/7天 时间范围切换

### 4. 折叠式状态显示

"当前状态"区域支持折叠/展开：

- 点击标题栏可折叠/展开状态卡片区域
- 显示当前监控目标数量徽章
- 默认展开状态
- 适合监控目标较多的场景，节省屏幕空间

### 5. 整体美化设计

#### 视觉优化
- 现代化的渐变色头部设计
- 圆角卡片和阴影效果
- 平滑的过渡动画
- 悬停效果增强交互体验
- 优化的颜色搭配和间距

#### 响应式布局
- 完美支持桌面、平板、手机等各种设备
- 自适应的卡片布局
- 移动端优化的字体大小和间距

#### 状态卡片改进
- 更清晰的成功/失败状态指示
- 优化的信息层级展示
- 更好的时间戳格式化
- 悬停时的卡片提升效果

## 配置示例

完整的配置文件示例：

```json
{
  "title": "我的网络监控",
  "description": "监控关键服务的网络连接状态",
  "targets": [
    {
      "addr": "8.8.8.8",
      "description": "Google DNS",
      "hide_addr": false
    },
    {
      "addr": "example.com",
      "description": "示例网站",
      "hide_addr": true
    }
  ],
  "ping_interval": 10,
  "web_port": 8081,
  "default_dns": ""
}
```

## 使用建议

1. **标题设置**: 建议使用简短明确的标题，如 "生产环境监控"、"网络质量监控" 等
2. **介绍文字**: 可以添加监控目的、注意事项等简短说明
3. **主题选择**: 长时间使用建议选择深色模式，更护眼
4. **目标选择**: 对比分析时建议选择 3-5 个目标，避免图表过于拥挤
5. **时间范围**: 查看实时状态用 1小时，趋势分析用 24小时或 7天

## 技术特性

- 使用 Bootstrap 5.3 现代化 UI 框架
- Chart.js 3.x 图表库
- 原生 JavaScript，无需额外依赖
- CSS3 动画和过渡效果
- LocalStorage 本地存储主题偏好
- 完全响应式设计
